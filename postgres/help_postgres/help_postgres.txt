-------------- AIDE DOCKER COMPOSE POSTGRE ---------------
construire le docker compose:
docker-compose -p postgre_streaming up -d

forcer a tout rebuild:
docker-compose -p postgre_streaming up --build -d

redemarrer le docker compose:
docker-compose -p postgre_streaming start

arreter le docker compose:
docker-compose -p postgre_streaming stop

supprimer le docker compose et docker associés:
docker-compose -p postgre_streaming down

recréer le docker compose:
docker-compose -p postgre_streaming down
docker-compose -p postgre_streaming up -d

TUTORIEL PREMIER FONCTIONNEMENT

* créer les containers
docker-compose -p postgre_streaming up -d
et aller sur
http://localhost:8081/

* indiquer user:
admin@admin.com
indiquer password:
password

* connecter postgre au panneau d'admin: (a été automatisé, ne plus faire)
cliquedroit sur servers->register->server
name= crypto_db
aller sur "Connection"
hostname/adress = postgres (doit être le nom du service sur le dockercompose)
port= 5432 (celui par défaut)
username = admin
password= password

-----------------------
* créer une table: (a été automatisé, ne plus faire)
sidebar a gauche, aller sur Servers > crypto-db > Databases > crypto_db > Schemas > public > Tables
clic droit et créer table
- OU -
aller sur barre de gauche gauche query tool workspace et selectionner la base postgre
écrire le contenu du fichier postgre_test.sql
